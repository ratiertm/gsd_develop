## 2. 전략 6종

### MA_CROSSOVER — 이동평균 골든크로스

```
인디케이터: EMA(3), EMA(10), RSI(7), Bollinger(10, 2.0σ)

진입 (AND):
  EMA(3) cross_above EMA(10)      # 단기 > 장기 전환
  RSI 35~65                        # 횡보 필터 (극단 제외)
  price > bollinger_middle          # 밴드 중심 위
  price > prev_low                  # 전일 저가 지지
  gap_pct < 2.0%                   # 갭업 추격 금지
  hour < 14                        # 마감 전 진입 금지
  kospi_pct > -1.0%               # 폭락장 차단
  prev_body_pct > 0.5%            # 전일 양봉일 때만

청산 (OR):
  EMA(3) cross_below EMA(10)
  RSI > 80
  price > prev_high                 # 전일 고가 도달 시 익절
```

### MACD_SIGNAL_CROSS — MACD 시그널 크로스

```
인디케이터: MACD(6,13,5), RSI(7), Bollinger(10, 2.0σ)

진입 (AND):
  macd_line cross_above macd_signal # MACD 골든크로스
  macd_histogram > 0                # 모멘텀 확인
  RSI 35~65                        # 횡보 필터
  price > bollinger_middle
  price > prev_low
  gap_pct < 2.0%
  hour < 14
  kospi_pct > -1.0%
  prev_body_pct > 0.5%

청산 (OR):
  macd_line cross_below macd_signal
  RSI > 80
  price > prev_high
```

### RSI_REVERSAL — RSI 과매도 반등

```
인디케이터: RSI(7)

진입 (AND):
  RSI < 20                         # 극단 과매도
  price > prev_low                  # 전일 저가 지지
  gap_pct > -2.0%                  # 급락 추격 금지
  hour < 14
  kospi_pct > -1.0%
  prev_upper_wick_pct < 3.0%       # shooting star 다음날 제외

청산 (OR):
  RSI > 80
  price > prev_high
```

### BOLLINGER_REVERSAL — 볼린저 하단 반등

```
인디케이터: Bollinger(10, 2.5σ), RSI(7)

진입 (AND):
  price < bollinger_lower           # 하단밴드 이탈
  RSI < 20                         # 극단 과매도
  price > prev_low
  gap_pct > -2.0%
  hour < 14
  kospi_pct > -1.0%
  prev_upper_wick_pct < 3.0%

청산 (OR):
  price > bollinger_middle          # 중심선 회복
  RSI > 85
  price > prev_high
```

### VWAP_BREAKOUT — VWAP 돌파

```
인디케이터: VWAP, RSI(7)

진입 (AND):
  price cross_above VWAP            # VWAP 상향 돌파
  RSI > 50                         # 상승 모멘텀
  price > prev_low
  gap_pct < 2.0%
  hour < 14
  kospi_pct > -1.0%
  prev_body_pct > 0                # 전일 양봉

청산 (OR):
  price cross_below VWAP
  RSI > 80
  price > prev_high
```

### GAP_REVERSAL — 갭하락 반등

```
인디케이터: RSI(7), VWAP

진입 (AND):
  gap_pct < -1.0%                  # 갭다운 -1% 이상
  price cross_above prev_low        # 전일 저가 회복
  RSI < 40
  9시~11시만                       # 오전 한정
  kospi_pct > -1.0%

청산 (OR):
  price > prev_close                # 전일 종가 회복 (갭 되메움)
  RSI > 70
  14시 이후
```

## 7. 핵심 발견 및 교훈

### 비용이 전략을 지배한다

- 왕복 0.21%~0.31%가 1분봉 변동폭(0.01~0.05%)보다 5~20배 큼
- **거래세 0.18%가 비용의 85%** — 전략으로 회피 불가
- 매매 1회당 평균 손익이 -4,000~-6,000원 (비용만으로)

### 필터는 효과적이지만 한계가 있다

- 시간/횡보/전일 데이터/일봉 패턴 필터로 **과매매 50~80% 감소**
- 하지만 남은 거래도 비용을 이기지 못함
- 필터를 강화하면 시그널 자체가 사라짐 (3분봉 + 강화 파라미터 → 대부분 0건)

### 횡보장에서 이기는 전략이 없다

- 2일 중 수익을 낸 건 RSI/Bollinger의 3/18 상승장 1건씩뿐
- 3/17 횡보장에서는 전 전략 전패
- 데이트레이딩의 근본 문제: **거래세를 이길 만큼 큰 일중 변동이 필요**

### 추세 전략 vs 역추세 전략

- MA/MACD (추세): 횡보장 과매매, 상승장에서도 노이즈에 취약
- RSI/Bollinger (역추세): 보수적이라 손실은 적지만 수익 기회도 제한적
- **역추세 전략이 데이트레이딩에 더 적합** (비용 대비 리스크 관리)

## 9. 향후 과제

1. **종목 선정 자동화** — 변동성 높은 종목을 당일 아침에 스크리닝
2. **보유 시간 연장** — 청산 조건 완화하여 수익 극대화 (비용 대비)
3. **수수료 무료 계좌** — 키움 비대면 이벤트 활용 시 왕복 0.18%
4. **멀티데이 백테스트** — 2일이 아닌 20일+ 데이터로 통계적 유의성 확보
5. **시장가/지정가 혼합** — 진입은 지정가(슬리피지 0), 손절은 시장가(확실한 체결)
6. **KOSPI 지수 실시간 필터** — collector에서 수집한 업종지수로 동적 차단
