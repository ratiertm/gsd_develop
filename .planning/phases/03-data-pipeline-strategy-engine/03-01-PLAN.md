    - BollingerBands(period=20, num_std=2.0): returns (upper, middle, lower), uses population stddev
    - VWAP: update_candle(high, low, close, volume), cumulative, reset() for daily reset
    - OBV: update(close, volume) -> int, cumulative, direction-based volume accumulation
    - Each indicator independently tracks its own state (no shared mutable state between instances)
