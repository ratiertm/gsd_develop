<feature>
  <name>PerformanceCalculator pure functions</name>
  <files>kiwoom_trader/backtest/performance.py, tests/test_performance.py</files>
  <behavior>
    Core metrics (all pure functions, no class state):

    1. calc_total_return(initial_capital, final_capital) -> float (percentage)
       - 10_000_000 -> 11_000_000 = 10.0%
       - 10_000_000 -> 9_000_000 = -10.0%
       - 10_000_000 -> 10_000_000 = 0.0%
       - initial_capital=0 -> return 0.0

    2. calc_max_drawdown(equity_curve: list[tuple[datetime, float]]) -> float (percentage)
       - [10M, 12M, 9M, 11M] -> MDD = (12M-9M)/12M*100 = 25.0%
       - Monotonically increasing -> 0.0%
       - Empty list -> 0.0%
       - Single point -> 0.0%

    3. calc_win_rate(trades: list[TradeRecord]) -> float (percentage)
       - Only counts SELL trades (BUY trades don't have realized P&L)
       - 3 sells: 2 profitable (pnl > 0), 1 loss -> 66.67%
       - No sell trades -> 0.0%

    4. calc_profit_factor(trades: list[TradeRecord]) -> float
       - SELL trades only. gross_profit / gross_loss
       - All profits, no losses -> float('inf')
       - All losses -> 0.0
       - No trades -> 0.0
       - gross_profit=300000, gross_loss=100000 -> 3.0

    5. calc_sharpe_ratio(daily_returns: list[float], risk_free_rate=0.035, trading_days=252) -> float
       - Uses annualized excess return / annualized std dev
       - daily_rf = risk_free_rate / trading_days
       - If std_ret == 0 -> 0.0
       - Fewer than 2 data points -> 0.0

    Additional metrics:
    6. calc_avg_pnl(trades) -> float: average pnl of SELL trades
    7. calc_max_consecutive_losses(trades) -> int: longest streak of SELL trades with pnl <= 0
    8. calc_total_trades(trades) -> int: count of SELL trades (round-trips)
    9. calc_avg_holding_period(trades) -> float: average bars between BUY and corresponding SELL (approx from trade timestamps)
    10. calc_daily_returns(equity_curve) -> list[float]: daily percentage returns from equity snapshots

    Aggregator:
    11. compute_all_metrics(result: BacktestResult) -> BacktestResult: fills all metric fields and returns the result
  </behavior>
