  <behavior>
    - BacktestEngine.__init__ takes strategy_config (list of strategy dicts), risk_config (RiskConfig), cost_config (CostConfig), initial_capital (int)
    - BacktestEngine.run(candles, on_progress) returns BacktestResult
    - Creates fresh StrategyManager + ConditionEngine per run (avoid stale state between runs)
    - Feeds candles one-by-one to strategy_manager.on_candle_complete()
    - BUY signal: validates position limits (max_positions, max_symbol_weight), calculates qty from capital * weight / price, applies buy cost
    - SELL signal: closes position, applies sell proceeds (commission + tax + slippage)
    - Per-candle SL/TP/trailing stop checks using RiskConfig percentages against close price
    - Trailing stop: tracks highest price since entry, triggers when close < highest * (1 - trailing_stop_pct/100)
    - Daily loss limit: if daily realized loss exceeds daily_loss_limit_pct of initial capital, skip further buys for that day
    - Forces close all positions at last candle price
    - Records equity snapshot after each candle
    - on_progress callback called with (current_index, total_candles)
    - Empty candle list returns BacktestResult with zero trades
  </behavior>
  <action>
    1. Create kiwoom_trader/backtest/backtest_engine.py:
       - class BacktestEngine:
         - __init__(self, strategy_configs: list[dict], risk_config: RiskConfig, cost_config: CostConfig, initial_capital: int = 10_000_000)
         - run(self, candles: list[Candle], on_progress: Callable[[int, int], None] | None = None) -> BacktestResult
         - Internal state per run: _capital (float), _positions (dict: code -> {qty, avg_price, highest_price}), _trades (list[TradeRecord]), _equity_curve (list), _daily_pnl (dict: date -> float)
         - Creates fresh StrategyManager(ConditionEngine()) and loads strategy_configs at start of run()
         - Do NOT pass risk_manager or paper_trader to StrategyManager -- backtest handles execution itself
         - StrategyManager.on_candle_complete returns signals; BacktestEngine processes them
         - _execute_buy(signal, candle): check max_positions and symbol weight, calc qty = int(capital * max_symbol_weight_pct/100 / calc_buy_cost(price, 1, cost_config)), deduct cost from capital, add to positions
         - _execute_sell(code, price, reason, candle): calc sell proceeds, compute pnl, add to trades, remove from positions, add to capital
         - _check_risk_triggers(candle): for each position matching candle.code, check SL (close <= avg_price * (1 + stop_loss_pct/100)), TP (close >= avg_price * (1 + take_profit_pct/100)), trailing stop (close <= highest * (1 - trailing_stop_pct/100)). Update highest_price if close > current highest.
         - _check_daily_loss(date): sum realized losses for this date, return True if exceeds limit
         - _close_all_positions(last_candle): sell all remaining at last_candle.close
         - _record_equity(timestamp): capital + sum(pos.qty * current_price for all positions)
         - _build_result(): construct BacktestResult with all collected data
